## 💬 Usage
`$ git clone https://github.com/aocattleya/hoge.git` <br>
`$ cd hoge` <br>
`$ bundle install` <br>
`$ rails db:create` <br>
`$ rails db:migrate` <br>
`$ rails s `<br>
`👉 http://localhost:3000 `
